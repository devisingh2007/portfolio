# 3D Technical Portfolio | Devisingh Rajput

A high-performance, responsive portfolio featuring advanced 3D animations, a technical skills pyramid, and optimized asset delivery. Built with React.js, Tailwind CSS, and Framer Motion.

## 🚀 Key Features

- **3D Skills Pyramid**: A custom-engineered triangular layout for technical skills with golden glow effects and hover-depth animations.
- **Dynamic Hackathon Section**: Responsive grid showcasing competition certificates with "Finalist (Last Round)" badges.
- **Advanced Glassmorphism**: Premium UI aesthetics with consistent theme-aware designs (Light/Dark mode supported).
- **SEO Optimized**: Fully configured meta tags, OpenGraph previews, and optimized static asset paths for production.

## 🛠️ Tech Stack

**Frontend:**
- React.js 19
- Tailwind CSS
- Framer Motion
- React Icons

**Development & Deployment:**
- Vite (Build Tool)
- Netlify (Hosting)
- Postman (API Testing)

## 📁 Repository Structure

```
├── public/                 # Static assets (PDF certificates, favicon)
├── src/
│   ├── assets/             # Project images and graphics
│   ├── components/         # Reusable 3D UI components
│   ├── data/               # Central data layer (projects, skills, hackathons)
│   ├── sections/           # Modular page sections (Hero, About, etc.)
│   └── index.css           # Global design system and 3D utilities
└── index.html              # Main entry point with SEO configurations
```
